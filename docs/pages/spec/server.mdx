# Server and Runtime

## Entry and Transport

- Runtime entry: `src/index.ts`
- Server factory: `src/server.ts`
- Transport mode: stdio-focused execution path (`MANTLE_MCP_TRANSPORT=stdio`)

## Registration Surface

The server exposes:

- tools (operational actions with schemas)
- resources (readable context endpoints)
- prompts (workflow guidance)

Detailed URI and method mapping is documented in [URLs and Interfaces](/spec/interfaces).

## Error Contract

Tool handlers use typed error payloads for predictable agent behavior:

- machine-readable `code`
- human-readable `message`
- actionable `suggestion`
- optional structured `details`

This contract is central for reliable LLM behavior in both direct usage and E2E assertions.

## Security-Critical Runtime Rules

- endpoint policy checks before outbound calls
- read-only SQL enforcement in indexer query path
- explicit timeout handling for external requests
- bounded payload handling for subgraph responses
