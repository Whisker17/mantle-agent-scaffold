# URLs and Interfaces

This page is the interface map for the current `mantle-mcp` implementation.

## MCP Request Interfaces

`src/server.ts` registers the following MCP interfaces:

| Interface | Purpose | Notes |
| --- | --- | --- |
| `listTools` | Discover callable tool contracts | Returns tool `name`, `description`, `inputSchema` |
| `callTool` | Execute one tool with structured arguments | Unknown tool returns typed `UNKNOWN_TOOL` error payload |
| `listResources` | Discover contextual resources | Returns URI catalog from `src/resources.ts` |
| `readResource` | Read one resource by URI | Unknown URI returns plain text `Resource not found` |
| `listPrompts` | Discover prompt templates | Returns prompt metadata from `src/prompts.ts` |
| `getPrompt` | Fetch prompt message templates | Unknown prompt throws `PROMPT_NOT_FOUND` |

## Resource URIs (`mantle://...`)

| URI | Backing Source | Purpose |
| --- | --- | --- |
| `mantle://chain/mainnet` | `CHAIN_CONFIGS.mainnet` | Canonical mainnet config (chain id, RPC, explorer, bridge, WMNT) |
| `mantle://chain/sepolia` | `CHAIN_CONFIGS.sepolia` | Canonical Sepolia testnet config and faucet URLs |
| `mantle://registry/contracts` | `src/config/registry.json` via registry loader | Trusted contract address resolution source |
| `mantle://registry/tokens` | `MANTLE_TOKENS` | Embedded token quick-reference list |
| `mantle://registry/protocols` | `MANTLE_PROTOCOLS` | DeFi protocol metadata and contract endpoints |
| `mantle://docs/network-basics` | `skills/mantle-network-primer/references/mantle-network-basics.md` | Network fundamentals reference for agents |
| `mantle://docs/risk-checklist` | `skills/mantle-risk-evaluator/references/risk-checklist.md` | Risk checklist reference before execution |

## Prompt Interfaces

| Prompt | Arguments | Purpose |
| --- | --- | --- |
| `mantle_portfolioAudit` | `wallet_address` (required), `network`, `scope` | Structured wallet audit workflow (balances, allowances, valuation, risk) |
| `mantle_mantleBasics` | none | Mantle ecosystem onboarding with chain/token/protocol basics |
| `mantle_gasConfiguration` | none | Gas-token semantics (MNT), fee-check workflow, RPC recommendations |

## Network URLs in Chain Config

| Network | URL Field | URL | Purpose |
| --- | --- | --- | --- |
| Mainnet | `rpc_url` | `https://rpc.mantle.xyz` | Default JSON-RPC endpoint for on-chain reads |
| Mainnet | `ws_url` | `wss://rpc.mantle.xyz` | WebSocket endpoint for subscription-capable clients |
| Mainnet | `explorer_url` | `https://mantlescan.xyz` | Human transaction/address explorer |
| Mainnet | `bridge_url` | `https://app.mantle.xyz/bridge` | Official bridge entry point |
| Sepolia | `rpc_url` | `https://rpc.sepolia.mantle.xyz` | Default Sepolia JSON-RPC endpoint |
| Sepolia | `explorer_url` | `https://sepolia.mantlescan.xyz` | Sepolia explorer |
| Sepolia | `bridge_url` | `https://app.mantle.xyz/bridge?network=sepolia` | Bridge link pinned to Sepolia |
| Sepolia | `faucet_urls[]` | Mantle/QuickNode/Thirdweb faucet URLs | Test MNT acquisition |

## External Service Endpoints and Env Interfaces

| Interface | Default/Example | Purpose |
| --- | --- | --- |
| `MANTLE_RPC_URL` | `https://rpc.mantle.xyz` | Override mainnet RPC endpoint used by client factory |
| `MANTLE_SEPOLIA_RPC_URL` | `https://rpc.sepolia.mantle.xyz` | Override Sepolia RPC endpoint |
| `MANTLE_TOKEN_LIST_URL` | `https://token-list.mantle.xyz` | Canonical token-list source used by `mantle_resolveToken` cross-check |
| `MANTLE_TOKEN_LIST_PIN_HASH` | `<sha256 or version>` | Pin token-list payload/version to block unexpected registry drift |
| `MANTLE_ALLOWED_ENDPOINT_DOMAINS` | `subgraph-api.mantle.xyz,example.com` | Domain allowlist for user-provided endpoint tools |
| `MANTLE_ALLOW_HTTP_LOCAL_ENDPOINTS` | `true/false` | Permit `http://localhost` only (for local debug); HTTPS still required otherwise |
| `E2E_SUBGRAPH_ENDPOINT` | `https://.../graphql` | Endpoint used by indexer GraphQL E2E scenario |
| `E2E_SQL_ENDPOINT` | `https://.../sql` | Endpoint used by indexer SQL E2E scenario |
| OpenRouter base URL | `https://openrouter.ai/api/v1` | Used when `E2E_LLM_PROVIDER=openrouter` |

## Tool-Level Endpoint Interfaces

| Tool | URL/Interface Input | Purpose |
| --- | --- | --- |
| `mantle_querySubgraph` | `endpoint` (GraphQL URL) | Execute GraphQL queries with timeout + response-size guardrails |
| `mantle_queryIndexerSql` | `endpoint` (SQL API URL) | Execute read-only SQL only; mutation keywords are blocked |
| `mantle_checkRpcHealth` | optional `rpc_url` | Verify endpoint reachability + chain-id match for target network |
| `mantle_probeEndpoint` | required `rpc_url`, optional `method` | Low-level probe for endpoint diagnosis |

## Allowed Probe Methods (`mantle_probeEndpoint`)

Only these JSON-RPC methods are allowed:

- `eth_chainId`
- `eth_blockNumber`
- `eth_getBalance`
