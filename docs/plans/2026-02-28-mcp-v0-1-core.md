# Mantle MCP v0.1 Core Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Ship the `v0.1-core` release from `specs/mcp-release-plan.md` with stdio transport, core read tools, core resources, tests, and docs alignment.

**Architecture:** Build a modular TypeScript MCP server with a shared tool contract and uniform typed error handling. Implement only `chain`, `registry`, `account`, and `token` tool modules for v0.1, backed by static config + on-chain reads through viem. Mirror required static resources and keep release-plan checkboxes synchronized with implemented scope.

**Tech Stack:** Node.js 18+, TypeScript (ESM), @modelcontextprotocol/sdk, viem, Vitest
